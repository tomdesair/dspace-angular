# Docker
## On Linux or OS X
1. `cd docker`
2. `./build.sh`
	- This only needs to happen the first time, and after something changes to `package.json`. 
3. `./run.sh`
	- run this (re)start the server 


## On Windows
1. `cd docker`
2. `build.bat`
	- This only needs to happen the first time, and after something changes to `package.json`. 
3. `cd ..`
4. `docker\run.bat`
	- run this (re)start the server
